## 📖 Sobre o Projeto

A **Casa Sandrissíma** é uma ONG dedicada à educação e ao desenvolvimento social de seus alunos. Este site foi desenvolvido como **Projeto Integrador** pelo grupo **HardCoders**, com o objetivo de:

- 🎓 **Organizar** o cadastro e o acompanhamento dos alunos
- 📣 **Divulgar** os projetos e iniciativas da ONG
- 🤝 **Conectar** doadores, associados e professores à causa

> *"Código com propósito: cada linha escrita aqui representa um passo a mais na missão da Casa Sandrissíma."*

## 🚀 Como Rodar Localmente

Não precisa de servidor nem instalação. Simples assim:

**1. Clone o repositório**
```bash
git clone https://github.com/hardcodersgroup/Casa-Sandrissima.git
```

**2. Acesse a pasta**
```bash
cd Casa-Sandrissima
```

**3. Abra no navegador**
```bash
# Basta abrir o arquivo index.html diretamente no seu navegador
# Ou, se tiver o VS Code com Live Server instalado:
code .
# → Clique com botão direito no index.html → "Open with Live Server"
```

✅ Sem dependências. Sem `npm install`. Só HTML, CSS e JS puros.
